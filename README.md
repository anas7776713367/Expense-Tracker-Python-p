# نظام إدارة الحسابات والمصروفات (Account Management System) 

برنامج بسيط ومبني بلغة بايثون يساعد المستخدمين على تسجيل مصروفاتهم اليومية، حفظها في ملفات CSV، وتحليلها عبر رسوم بيانية تفاعلية.

##  المميزات
* **إضافة العمليات:** تسجيل المبلغ، الفئة، والوقت تلقائياً.
* **تخزين البيانات:** حفظ كافة البيانات في ملف `CSV` لضمان عدم ضياعها.
* **التقارير:** عرض قائمة المصروفات وحساب الإجمالي بضغطة زر.
* **التحليل البصري:** توليد رسوم بيانية (Pie Chart & Bar Chart) لفهم توزيع المصروفات.

## المكتبات المستخدمة
يستخدم المشروع المكتبات التالية:
* **Pandas:** لمعالجة البيانات وجداول الـ CSV.
* **Matplotlib:** لإنشاء الرسوم البيانية.
* **NumPy:** للعمليات الحسابية.
